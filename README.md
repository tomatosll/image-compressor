# 图片压缩工具

一个现代化的在线图片压缩工具，支持批量处理，提供原图与压缩后对比功能。

## 功能特点

- 支持 JPG、PNG、WebP、GIF 格式的图片上传与压缩
- 可设置压缩质量（高/中/低或自定义百分比 1-100%）
- 显示原始图片与压缩后图片的对比（尺寸、大小、预览图）
- 支持批量处理，最多同时处理 20 张图片
- 压缩完成后提供单独下载和批量下载功能
- 响应式设计，支持手机和桌面设备
- 现代简约风格，绿色主题

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Node.js + Express
- **图片处理**: Sharp

## 安装与运行

### 1. 安装依赖

```bash
cd image-compressor/server
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 访问应用

打开浏览器访问 http://localhost:3000

## 使用说明

1. **上传图片**: 拖放图片到上传区域，或点击选择文件
2. **设置压缩参数**:
   - 选择压缩质量（高/中/低/自定义）
   - 选择输出格式（保持原格式或转换）
3. **开始压缩**: 点击"开始压缩"按钮
4. **查看结果**: 对比原图和压缩后的图片
5. **下载**: 单独下载或批量下载所有压缩后的图片

## 注意事项

- 单个文件最大支持 50MB
- 一次最多上传 20 张图片
- 所有图片在服务器上临时存储，1 小时后自动删除
